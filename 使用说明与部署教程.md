# 摩耶重炮 语音站 · 使用说明与部署教程

一个可以自由选择、播放摩耶重炮(マヤノトップガン)角色语音的网页,带天空飞行动态背景、中日双语标题、分类筛选和搜索。

---

## 一、文件结构

```
摩耶重炮/                     ← 网站根目录(把这个文件夹整个部署即可)
├── index.html              ← 网页本体(动态背景 + 播放器)
├── data.js                 ← 语音数据(标题/日文/分类,由脚本生成)
├── manifest.json           ← 同样的数据(标准 JSON,备份/参考用)
├── audio/                  ← 215 个音频,已改成安全英文名 001.mp3 … 215.mp3
│   ├── 001.mp3
│   └── …
├── 使用说明与部署教程.md      ← 本文件
└── _build/                 ← 构建脚本(部署时可不上传,留着方便以后修改)
    ├── filelist.txt        ← 原始日文文件名顺序(对照用)
    ├── translations.mjs    ← 中文翻译 + 分类
    └── build.mjs           ← 重新生成 data.js / manifest.json 的脚本
```

> 音频原来的日文文件名(如 `いっくよー！.mp3`)已完整记录在 `manifest.json` 和 `data.js` 的每条 `orig`/`jp` 字段里,随时可对照或还原。

---

## 二、本地预览

直接**双击 `index.html`** 就能在浏览器里打开使用(数据是用 `<script src="data.js">` 载入的,不受 `file://` 限制)。

如果个别浏览器音频无法播放,用一个本地小服务器打开更稳妥:

```bash
# 在 摩耶重炮 文件夹里执行(需装了 Node)
npx serve .
# 然后浏览器打开提示的 http://localhost:3000
```

## 三、部署到 GitHub Pages(让别人也能访问)

这样会得到一个 `https://你的用户名.github.io/仓库名/` 的公开网址,任何人点开就能用。

### 方式 A：网页上传(最简单,零命令)

1. 注册并登录 [github.com](https://github.com)。
2. 右上角 **+ → New repository**;仓库名例如 `maya-voice`;选 **Public**;点 **Create repository**。
3. 进入空仓库页面,点 **uploading an existing file**(或 Add file → Upload files)。
4. 把 `摩耶重炮` 文件夹里的内容拖进去:
   - **必传**:`index.html`、`data.js`、`manifest.json`,以及整个 `audio/` 文件夹。
   - `_build/` 和本说明文件可传可不传(不影响运行)。
   - 直接把 `audio` 文件夹拖到上传框,GitHub 会保留子目录。
5. 下方点 **Commit changes** 等待上传完成(44MB,稍等一会)。
6. 仓库页 **Settings → Pages**:
   - **Source** 选 `Deploy from a branch`;
   - **Branch** 选 `main`、目录选 `/ (root)`;点 **Save**。
7. 等 1–2 分钟刷新,Pages 里会显示网址 `https://你的用户名.github.io/maya-voice/`。把这个链接发给别人即可。

### 方式 B：命令行(Git)

```bash
cd ~/Desktop/项目/摩耶重炮
git init
git add index.html data.js manifest.json audio
git commit -m "摩耶重炮语音站"
git branch -M main
git remote add origin https://github.com/你的用户名/maya-voice.git
git push -u origin main
# 之后同样在 Settings → Pages 里开启 main / root
```

> 注意:`项目` 文件夹本身已经是一个 Git 仓库(含多个子项目)。上面的 `git init` 会在 `摩耶重炮` 子文件夹里新建一个独立仓库,只部署本站,互不影响。

## 四、以后想修改内容

改完翻译或分类后,重新生成数据即可:

```bash
cd ~/Desktop/项目/摩耶重炮
node _build/build.mjs        # 重新生成 data.js 和 manifest.json
```

- **改翻译/分类**:编辑 `_build/translations.mjs`(每条的 `cn` 是中文、`cat` 是分类键)。
- **换背景/配色**:编辑 `index.html` 顶部 `:root{}` 里的颜色变量,或 `.sky` 相关的动画。
- **加/删音频**:较麻烦(涉及重新编号),需要时告诉我来处理。

分类键:`greet 问候` · `daily 日常` · `train 训练` · `race 出走` · `win 胜利` · `lose 失落` · `growth 成长` · `story 剧情` · `event 活动` · `live 演出` · `sys 系统`。

## 五、小提示

- 音频与文件夹都用的是英文名,GitHub Pages 上不会出现乱码或加载失败。
- 网页支持:🔍 搜索(中/日文)、分类筛选、🎲 随机播放、上一条/下一条、自动连播、进度拖动;快捷键 **空格**播放/暂停、**←/→** 切换。
