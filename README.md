# 摩耶重炮 语音站 🛩️

[赛马娘](https://zh.moegirl.org.cn/赛马娘_Pretty_Derby) 角色 **[摩耶重炮（マヤノトップガン）](https://zh.moegirl.org.cn/摩耶重炮)** 的同人语音收藏站。

**[🔗 zhoumornnig.github.io/maya-voice](https://zhoumornnig.github.io/maya-voice/)**

## 功能

- 🎧 **215 条日文语音**,每条配中文翻译,按 11 个分类整理（问候/日常/训练/比赛/胜利/失落/成长/剧情/活动/演出/系统）
- 🔍 中/日文搜索 + 分类筛选 + 随机播放 + 自动连播
- 🖼 **74 张角色动图画廊**,分页浏览 + 点击放大,视口内自动播放
- 🎨 天空飞行主题动态背景（飘云、樱花瓣、随机出现的角色动图、可点击召唤）
- 🎚 可拖动/可收起的悬浮播放器 + 音量调节
- 📱 响应式,手机/电脑都能用
- ⚡ 纯静态,无框架无依赖,部署在 GitHub Pages

## 技术

- 纯 HTML + CSS + 原生 JS,零依赖零框架
- 动图用 sharp 批量压缩+减帧（68 MB → 5.7 MB,缩小 92%）
- IntersectionObserver 做视口内播放,懒加载 + content-visibility 优化性能
- fetchPriority 低优先级标记,避免动图抢占音频带宽
